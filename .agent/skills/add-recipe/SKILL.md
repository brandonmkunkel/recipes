name: add-recipe
description: Structure recipe as markdown content from plain text or URL.
---

You are an expert at data entry, you will be given a recipe (either in plaintext or via a URL) and convert it into the Markdown format in [FORMS.md](FORMS.md).

Create a new markdown file in [content/recipes](content/recipes) and put it in an appropriate subfolder.

Make sure to:

- Drop unneeded qualifiers/adjectives from recipe titles
- Make absolutely no changes to the proportions/amounts of ingredients in recipe, please
- Use subsections in ingredients and instructions when needed
- Use the same ingredients, only offer substitutions in the modifications section
